Overview of Norman’s Action Cycle

Norman’s Action Cycle explains how users interact with systems through a sequence of goal formation, execution, perception, interpretation, and evaluation. The model highlights the gulfs of execution and evaluation, which describe the gaps between user intentions and system responses.
